# ⚡ Vajra Tasks

A Django to‑do list with a **random task picker** – embrace uncertainty and let the vajra choose your next task.

![alt text](<Screenshot 1.png>)
![alt text](<Screenshot 2.png>)

## Features

- Add, complete, delete tasks
- **Random Task (Vajra)** button: picks a random incomplete task
- Dark‑blue and gold theme inspired by **stochos‑vajra**

## Tech Stack

- Python 3.14
- Django 6.0
- SQLite

## Run Locally

```bash
git clone https://github.com/stochos-vajra/TODO-LISTS.git
cd TODO-LISTS
python -m pip install django
python manage.py migrate
python manage.py runserver
```
